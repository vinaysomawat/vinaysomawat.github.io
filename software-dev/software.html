Games/Applications

Killer Sheep

Game featuring Prof. David Bernstein of James Madison University, controlled by the user, and a menacing herd of sheep.
The goal of this game is to direct Prof. Bernstein safely into a paddock without getting trampled by sheep.
Once in the paddock, the sheep will follow passively and the user will have won the game.
Otherwise, if the sheep get to Bernstein, the user will have lost the game.
Technologies Used: Java

TAP Scheduler

Online application for the Teaching Analysis Poll (TAP) consultation service at James Madison Univerisity.
Provides an efficient system for professors and consultants to sign up for sessions.
Technologies/Languages Used: HTML/CSS/JavaScript, Java, JSP
